随笔——2000-2023产品手绘风格迭代

Created2023-09-02|Updated2024-05-18
|Word count:2.2k|Reading time:7min|Post View:|Comments:

开篇声明，这不是一篇严谨的探究报告或论文，仅是一篇随笔，观点仅限个人，但欢迎大家参与讨论，我将用四篇博客完成这个随笔。

Y2K


        

        

Y2K设计风格主要流行于2000年，笼统来说，Y2K指的是千禧年前后的流行文化，它的巅峰时期是从1993年到2003年之间，乐观主义和技术乌托邦主义是它的核心思想。主要的核心是科技感，和当时人类在当时环境下对未来科技的发展可能产生出来某些事物的猜想所创造出来的，在当时互联网刚刚兴起，科技电器产业开始发展的大背景下，人们对科技感的兴趣十足，但同时互联网的快速而混乱的发展引发了人们对安全问题的担忧。于是以电脑互联网ui和电子产品为主，形成了一个鲜明独特的设计风格，这个风格也陆续延伸影响到了当时的时尚和设计领域，成为了当时的文化标签，立体感，科技感，鲜艳的颜色，充满未来感的设计可以被理解为这种美学的核心。




这种美学的产生背景与当时的科技环境背景也是息息相关的，没有过高分辨率的电脑，这种低清晰度的设计也成为了当时的主流。


清水吉治
清水吉治是日本著名的工业设计师,被称为“设计之神”。他1928年出生于日本长野县,1950年毕业于东京艺术大学。活跃于2000年左右，以下是老师的手绘作品，明快的风格使得作品观感偏向锐利，因此尽管分辨路不高，依然能够获得很好的观感。
画风严谨、用色细腻，物体的塑造一丝不苟，日本老一辈肯定而扎实的态度不仅暗藏在言行中，也一以贯之地出现在他的画面里。色粉作为他最常用的上色材料，在使用前，用量、混合程度、上色范围等顺序讲究考虑周全，作画时一步到位，不容许多次叠涂和覆盖修改，需要格外的耐心与仔细。正因如此，也将他这种特点诚实地在纸上展现出来：画面整洁工整，有很强的辨识度。


我们可以从画面中看出大量Y2K风格的元素，包括但不限于大量的金属镜面反光与强烈的科技感。
但显而易见的，随着老师的退休，Y2K在2004年迎来暂时的结束。而它的继任者，是那个几乎每个人都见过，霸占网络图片十余年的Frutiger Aero.


Frutiger Aero

        

            

        

        


这首音乐并没有正式的标题，也不知道它的来源和作者，唯一能知道的，就是这首音乐的文件名字叫做 title.wma，储存在系统的 C:\Windows\oobe 目录下。在 Windows XP 中，微软让系统在第一次启动的播放这首音乐.
这首音乐轻快愉悦，让 Windows XP 在第一次启动的时候能够营造轻松的氛围，让原本冰冷的操作系统显得更加人性化。
的确，很多人在第一次安装系统的时候都没有预料到首先遇到的是这一首音乐，让人觉得惊喜，甚至有一些人迫不及待地想要将这首音乐提取出来多播放几次。由于 Windows XP 超长的寿命和无敌的市场占有率，这首音乐就这样一直活在很多人的电脑里，十多年。
直到2014年微软宣布完全停止 Windows XP 所有支持，宣布 Windows XP 的死亡，很多人开始制作一些视频缅怀 Windows XP 与电脑前的各位相伴的时光，他们的视频常常就会加上 title.wma 这首音乐，但没人知道这首音乐叫什么名字。
显然，title.wma 已经成为 Windows XP 的缩影，而微软，启发了一个时代的美学，这个时代的美学，叫做Frutiger Aero。
————————————————————分割线——————————————————
以下解释截取自美学Wiki跳转地址


Frutiger Aero (也被称为Web 2.0 Gloss) 是大致在2004到2013年间流行在广告、全景图片库和技术领域的一种设计风格和美学，它承接着Y2K时代的结束并与McBling，Surf Crush和ElectroPop 08等美学时期重叠。它的特征在于它对拟物化，光泽纹理，多云的天空，热带鱼，水，气泡，玻璃，镜头光晕，杂乱的图案，“人文精神”，毛玻璃，焦外成像，Frutiger字体，极光等元素和明亮，鲜艳的颜色（通常是绿色和蓝色）的运用。


这种美学在当时没有被命名，更多与特定的风格联系（Windows 7的Aero主题，iOS的拟物化图标，游戏镜之边缘（Mirror’s Edge）和孢子（Spore）的视觉效果）而不是更广泛地被认为是一种美学。有些人在这种美学是主流时将其称为”Web 2.0 Gloss”，但这很大程度上只形容了Frutiger Aero中的光泽元素而不是整个美学。术语Frutiger Aero于2017年被消费者美学研究所（Consumer Aesthetics Research Institute）的Sofi Lee所创造，于2022年开始在网上快速传播，再次引起了人们对它的兴趣。Frutiger Aero包含 很多亚种和相关美学，包括但不限于Frutiger Metro，Technozen，Frutiger Eco和Helvetica Aqua Aero。
————————————————————分割线——————————————————


Helvetica Aqua Aero 是流传最为广泛的一个分支，多使用与海洋和沙滩图像，
这些图像包括水生生物，气泡，水，水系颜色及其它海洋关键元素，以及Frutiger Aero常见元素，
同时具备了Y2K风格，比如光泽，未来主义，三维渲染物件，00年代早期图像和渐变色。
但实际上，Helvetica Aqua Aero只是从Y2K到Frutiger Aero的过渡，流行时间也局限在了2004-2013年。


Frutiger Aurora出现在00年代早期，即Y2K时代的末期，在10年代早期仍维持有一定流行度。
Frutiger Aurora是一种注重极光视效的Frutiger Aero亚种；极光是一种主要在高纬度地区（北极和南极附近）出现在天空中的自然光，展现出像帷幔、射线、螺旋或闪烁物的明亮动态光效，覆盖整个天空。
如你所见，发行于2004年的Windows vista系统的默认壁纸，直接使用了Frutiger Aurora风格的壁纸，然而，是微软公司创造出了Frutiger Aurora这个亚种，还是Frutiger Aurora在当时已经流行并被微软所采纳，目前尚不可考证。

Frutiger Aero与卡尔刘
刘传凯作为新一代的设计师，笔触精简，画面突出思维过程，可以促进设计师彼此间设计想法的有效沟通；面向设计师和学生时，则是对学习概念构思以及设计手绘有指导作用。
刘传凯的《产品创意设计》是2005年中国青年出版社出版的图书，（刚好与Frutiger Aero的时间撞上）具有很明显的个人风格的同时，也具有大量的Frutiger Aero的风格。
比如相对圆润的光影，具有未来主义色彩的造型，大量运用运动与环保的主题，以及玻璃质感的表现。
显然，没有了Y2K时期的锐利光影与严谨的造型，产品却多了律动感，从手绘到产品实物，造型走向了圆润，曲线曲面被大量使用，这是一次成功的迭代。


在Frutiger Aero的后期（2013年），网络上大量的“英语课件式”的图片已然引起了消费者的乏味，人们迫切需要一个“破局者”打破到处都是的Frutiger Aero，而这个“破局者”，又是微软。
这次，是“扁平化设计”（flat design）与“产品设计板绘”.
